# Mac OS

## Usage
### FAQs
* 在文件选择框中显示隐藏文件

    使用 `CMD + >` 切换隐藏文件的显示.
    
    
    
## 字体
使用 `SF Mono` 字体, Fontsize 设为 `14`, Typeface 设为 `Light`. 如果没有 Typeface 选项可以直接将字体设置为 `SFMono-Light`.
安装 `SF Mono` 字体:
```shell
$ cp -R /Applications/Utilities/Terminal.app/Contents/Resources/Fonts/*.otf ~/Library/Fonts/
```
