# Shell

## CLI
### FAQs
* 将多条 Shell 命令合并执行:
    * 在当前 shell 中执行: `(cmd1,cmd2,cmd3...)`.
    * 在子 shell 中执行: `{cmd1,cmd2,cmd3...}` 或使用 `sh -c` 命令.
