# 程序设计

## 编译
### JIT
即时编译(Just-in-time compilation).
通常, 程序有两种运行方式: 静态编译和动态解释. 静态编译的程序在执行前全部被翻译为机器码, 而解释执行的则是一句一句边运行边翻译.
即时编译器则混合了这两者, 一句一句编译源代码, 但是会将编译过的代码缓存起来以降低性能损耗. 相对于静态编译代码, 即时编译的代码可以处理延迟绑定并增强安全性.
