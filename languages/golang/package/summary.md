# Package

## GOModule
### GOPROXY
如果设置了该环境变量, 下载源代码时将会通过这个环境变量设置的地址来下载. [goproxyio/goproxy](https://github.com/goproxyio/goproxy) 项目提供了一个公开的代理地址:
```shell
export GO111MODULE=on
export GOPROXY=https://goproxy.io
```
